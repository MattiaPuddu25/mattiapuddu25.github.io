Templates LaTeX



Over the years, I have gained considerable experience in using LaTeX, and I now systematically rely on these templates for books, articles, and beamer presentations. These templates have been tested on both TeXPad and Overleaf.

In the latest version (06/2025), several updates were made: the preamble is now minimal, and the parts that need to be modified by the user are contained in separate files and are clearly marked. Everything else, in principle, should not be changed unless the user knows exactly what they are doing.

All templates, once compiled, display in the PDF file the various commands and features that I have added. In the case of the beamer template, some specific functionalities of the beamer class are also illustrated.
